================================================================================
                              PROJECT OVERVIEW
================================================================================

VisionBoard Pro is a full-stack Business Vision Board application for
entrepreneurs and business leaders to create, manage, and track measurable
business goals and strategic plans. This is an execution-focused, data-driven
CEO-level planning tool built on modern business frameworks.

MISSION:
Help entrepreneurs define their business vision, set strategic goals, track
progress, and execute with precision using proven business frameworks.

================================================================================
                              TECH STACK
================================================================================

FRONTEND:
  - React.js 18 + Vite (Build tool)
  - Tailwind CSS (Styling)
  - React Router DOM (Navigation)
  - React Context API (State Management)
  - Axios (HTTP Client)

BACKEND:
  - Node.js + Express.js
  - MongoDB (Database)
  - Mongoose (ODM)
  - JWT (Authentication)
  - bcryptjs (Password hashing)

AI INTEGRATION:
  - OpenAI API (AI coaching, suggestions, analysis)

DEPLOYMENT:
  - Vercel (Frontend)
  - Render/Railway/Heroku (Backend)
  - MongoDB Atlas (Database)

================================================================================
                              CORE FEATURES
================================================================================

1. AUTHENTICATION SYSTEM
   - User registration with secure password hashing
   - JWT-based authentication
   - Protected routes for authenticated users
   - Admin-only routes for user management
   - Session management with token refresh
   - User impersonation (admin feature)

2. VISION BOARD MODULE (20+ Strategic Sections)
   - Company Overview, Core Purpose, Vision, Mission
   - Brand Promise, Core Values, BHAG (Big Hairy Audacious Goal)
   - Vivid Description, SWOT Analysis
   - Strategic Priorities with WWW Actions
   - 3-Year Strategy Map
   - SMART Goals with OKR support
   - Quarterly Execution Plan
   - Revenue Model & Targets (FP&A Dashboard)
   - Organizational Structure (FACe/PACe Charts)
   - SOP Roadmap, Automation & Systems
   - KPI Dashboard
   - Risk Management
   - Collaboration Hub
   - Strategy Summary

3. PROGRESS TRACKING
   - Real-time progress calculation across all sections
   - Visual progress bars at module and overall levels
   - Data-driven progress (based on actual content)
   - Monthly update tracking

4. PDF EXPORT SYSTEM
   - Full Vision Board export
   - One-Page Strategy Summary
   - Professional formatting

5. AI-POWERED FEATURES
   - AI suggestions for goal alignment
   - Risk scenario simulation
   - "Ask Patrick" AI coaching
   - Smart recommendations

6. ADMIN DASHBOARD
   - User management (CRUD)
   - Analytics dashboard
   - Platform-wide metrics
   - User activity monitoring

================================================================================
                              API ENDPOINTS
================================================================================

AUTHENTICATION:
  POST /api/auth/register
  POST /api/auth/login
  GET  /api/auth/me
  PUT  /api/auth/profile
  POST /api/auth/logout

VISION BOARDS:
  GET    /api/visionboards
  GET    /api/visionboards/:id
  POST   /api/visionboards
  PUT    /api/visionboards/:id
  DELETE /api/visionboards/:id
  PUT    /api/visionboards/:id/section/:sectionName

STRATEGY SHEET:
  GET /api/visionboards/:id/strategy
  PUT /api/visionboards/:id/strategy/:sectionName

PROGRESS:
  GET  /api/progress/:visionBoardId
  POST /api/progress/:visionBoardId/monthly

AI FEATURES:
  POST /api/ai/suggestions
  POST /api/ai/chat
  POST /api/ai/analyze

ADMIN:
  GET /api/admin/users
  GET /api/admin/analytics

================================================================================
                           DEVELOPMENT COMMANDS
================================================================================

ROOT LEVEL:
  npm run dev          - Start both frontend and backend
  npm run server       - Start backend only (port 5000)
  npm run client       - Start frontend only (port 5173)
  npm run install:all  - Install all dependencies
  npm run build        - Build frontend for production
  npm run start        - Start production server

================================================================================
                           ENVIRONMENT VARIABLES
================================================================================

SERVER (.env in /server):
  PORT=5000
  NODE_ENV=development
  MONGODB_URI=mongodb://localhost:27017/visionboard-pro
  JWT_SECRET=your-super-secret-jwt-key
  JWT_EXPIRE=7d
  OPENAI_API_KEY=your-openai-api-key

CLIENT (.env in /client):
