================================================================================
                    BEST PRACTICES FOR AI PROMPTS
================================================================================

DO:
  ✓ Include specific file paths when known
  ✓ Mention which module/feature you're working on
  ✓ Provide context about current behavior
  ✓ Reference similar existing implementations
  ✓ Specify which files the AI should read first
  ✓ Ask the AI to read context files before helping

DON'T:
  ✗ Ask vague questions without context
  ✗ Assume the AI remembers previous sessions
  ✗ Request changes without specifying files
  ✗ Ask to "fix everything" - be specific
  ✗ Forget to mention if it's frontend or backend

================================================================================
                    COMMON PATTERNS IN THIS PROJECT
================================================================================

When asking about:

AUTHENTICATION:
  Relevant: AuthContext.jsx, authController.js, auth.js (middleware)
  Pattern: JWT in localStorage, ProtectedRoute wrapper

VISION BOARDS:
  Relevant: VisionBoard.js (model), StrategySheetManager.jsx
  Pattern: Nested schema with completed/data structure

API CALLS:
  Relevant: services/ folder
  Pattern: async/await with try-catch, { success, data, message } response

COMPONENTS:
  Relevant: components/common/ for UI, components/[feature]/ for specific
  Pattern: Functional components with hooks, PropTypes not used (implicit)

STYLING:
  Relevant: Tailwind classes
  Pattern: Utility classes, dark: prefix for dark mode
